# Instalation

1. Run docker-compose build
2. Run docker-compose up -d

If is wanted to run without docker
1. Copy .env.example and paste it with the name .env
2. Replace for the right values. It is recommended to use MySQL or Postgres
3. composer install
4. npm run dev
5. php artisan db:seed
6. php artisan serve
7. Check the site on localhost:8000
